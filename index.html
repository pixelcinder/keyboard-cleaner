How it works

Up and running in minutes

Grant permission once, then it's one tap away every time you need it.

1

Grant Accessibility Permission

On first launch, grant one-time Accessibility permission in System Settings. This lets the app block keystrokes — nothing else is accessed.

2

Lock your keyboard

Click the Lock button or press ⌃⌘L from anywhere on your Mac. Every key is blocked instantly.

3

Clean freely

Clean every key freely. Nothing gets through — no accidental input, no opened apps, no typed text.

4

Unlock with Touch ID

One finger on Touch ID and you're back. Or use the on-screen PIN pad with your mouse — no keyboard needed.

Total keyboard block

Every keystroke — letters, shortcuts, function keys — is intercepted before it reaches any app. System-wide and instant.

How to install

Install it like this:

  1. Download the .dmg and open it.
  2. Drag Keyboard Cleaner into your Applications folder.
  3. Open Keyboard Cleaner.
  4. Allow Accessibility access when prompted.
