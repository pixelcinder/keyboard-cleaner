## Product Overview

Keyboard Cleaner is built for one specific job: letting you clean your Mac keyboard without triggering shortcuts, typing into documents, or sending accidental commands to the apps around you.

The app uses a global event tap to stop keyboard input before it reaches your apps. When the keyboard is locked, you can choose a full-screen overlay for maximum confidence or a compact floating view for a quieter utility feel. Unlocking stays deliberate and device-aware: Touch ID where available, PIN when configured, and password fallback when needed.

It is a focused utility, but it is treated like a real Mac app. That means native window behavior, accessibility support, menu bar control, onboarding, diagnostics, and a calm visual design that feels at home on current macOS.

## Help / Support Copy

### What does Keyboard Cleaner do?

Keyboard Cleaner temporarily locks your Mac keyboard so you can clean it without sending keystrokes to apps.

### How do I start a cleaning session?

Open Keyboard Cleaner and click `Lock Keyboard`, or use the global shortcut `Control` + `Command` + `L`.

### How do I unlock?

Unlock methods depend on your Mac and settings:

- Touch ID is used when available
- PIN unlock is available if you set a PIN in Settings
- Password fallback is available when biometric unlock is not possible or fails

### Why does the app need Accessibility access?

macOS requires Accessibility permission for apps that intercept and block global keyboard input. Without that permission, Keyboard Cleaner cannot lock the keyboard.

### What if locking does not work?

Open the app’s Diagnostics view and confirm:

- Accessibility permission is granted
- The event tap is active
- Your expected unlock method is available

If permission was just changed, relaunch the app and run a quick lock test.
